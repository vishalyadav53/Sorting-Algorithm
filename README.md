# Sorting Algorithms in Python



This repository contains implementations of popular sorting algorithms using Python.

## Algorithms Included

* Bubble Sort
* Selection Sort
* Insertion Sort

## Folder Structure

```text
Sorting/
├── BubbleSort.py
├── selectionSort.py
└── insertion Sort.py
```

## Description

### Bubble Sort

Repeatedly compares adjacent elements and swaps them if they are in the wrong order.

### Selection Sort

Finds the minimum element from the unsorted portion of the array and places it at the correct position.

### Insertion Sort

Builds the sorted array one element at a time by inserting each element into its correct position.

## Language Used

* Python

## How to Run

```bash
python BubbleSort.py
python selectionSort.py
python "insertion Sort.py"
```
